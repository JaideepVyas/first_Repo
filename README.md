Online Exam Management System (Django)

A Django web application for managing user accounts, exams, and results.
This project shows how multiple Django apps work together in a single web application.

Features:-

User registration and login (accounts)

Creating and managing exams (exam, examapp)

Displaying exam results (results)

HTML templates and static files (CSS/JS)

first_Repo/
│── accounts/      → User management
│── exam/          → Exam logic
│── examapp/       → Exam features
│── results/       → Result handling
│── templates/     → HTML files
│── static/        → CSS / JS
│── manage.py
│── db.sqlite3
